# Rahul Dodke — Portfolio

A single-file, zero-build portfolio. Works on GitHub Pages with no configuration needed.

**Live:** https://RDrahul123.github.io/Portfolio

## Deploy in 3 steps

1. Push this repo to GitHub (repo name: `Portfolio`)
2. Go to **Settings → Pages → Source** → select **GitHub Actions**
3. Push to `main` — live in ~60 seconds ✅

No Node.js, no npm, no build step. Just a single `index.html`.
