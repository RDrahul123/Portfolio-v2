Quick Guide
How the back office works

1. Use the sidebar to navigate to the section you want to edit.

2. Make your changes in the forms — everything is editable.

3. Click Save Changes to apply. Changes are stored in your browser and shown live on the portfolio.

4. Use Export JSON to back up all your data, and Import JSON to restore it on a new device.

⚠ Since this runs on GitHub Pages (no server), data is stored in your browser's localStorage. Always export a backup!
